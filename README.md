# AmanPlay 🎮🛡️
**Intelligent Cyberbullying Detection for Arabic Gaming Communities**

## 📌 Project Overview
AmanPlay is an AI-powered system designed to create a safer environment for Arabic-speaking gamers. It monitors and analyzes voice chats in real-time to detect toxicity and cyberbullying, bridging the gap that traditional keyword-based filters fail to address.

## ⚙️ Technical Architecture
The system follows a **Client-Server architecture** integrated with advanced AI models:
- **Frontend:** Built with **HTML/CSS** for a responsive dashboard.
- **Backend:** Powered by **Python** to manage data flow and API integrations.
- **AI Models (Elm Services):**
  - **Elm-ASR:** For high-accuracy Speech-to-Text conversion.
  - **Nuha-2.0:** For deep analysis of Arabic text and toxicity detection.

## 🚀 Key Features
- **Real-time Processing:** Instant detection of toxic behavior.
- **Dialect Awareness:** Deep understanding of various Arabic dialects and cultural context.
- **Seamless Integration:** Designed to work with various gaming platforms via APIs.

## 🛠️ Tech Stack
- **Language:** Python
- **Web:** HTML, CSS
- **APIs:** Elm Cloud Services (ASR & Nuha)
